# goapi
Simple REST Api with MongoDB and gorilla/mux

.env file
```
server_host=api:8080
mongo_user=root
mongo_password=password
mongo_host=db
mongo_port=27017
```

mongo.evn file

```
server_host=api:8080
mongo_user=root
mongo_password=password
mongo_host=db
mongo_port=27017
```
the server actually runs on localhost:8080
